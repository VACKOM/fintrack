# FinTrack

A personal finance management application built with the MERN stack.

## Features

- User Authentication
- Account Management
- Income Tracking
- Expense Tracking
- Budget Management
- Dashboard Analytics
- Financial Reports

## Tech Stack

### Frontend

- React
- React Router
- Axios
- Recharts
- CSS Modules

### Backend

- Node.js
- Express
- MongoDB
- Mongoose
- JWT Authentication

🚧 Currently in active development.
